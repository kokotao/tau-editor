## 应用功能介绍

### 核心功能

#### 📄 文件管理
- **新建文件**: 创建空白文档或从模板开始
- **打开文件**: 支持多种文件格式，自动检测编码
- **保存文件**: 快速保存，支持另存为
- **最近文件**: 快速访问最近编辑的文件

#### ✏️ 文本编辑
- **基础编辑**: 剪切、复制、粘贴、撤销、重做
- **多光标编辑**: 同时编辑多处内容
- **代码折叠**: 折叠/展开代码块
- **自动缩进**: 智能缩进和对齐

#### 🔍 搜索与替换
- **快速查找**: 在当前文件中搜索文本
- **全局搜索**: 在多个文件中同时搜索
- **正则表达式**: 支持高级正则匹配
- **替换功能**: 批量替换文本

#### 🎨 语法高亮
- **50+ 编程语言**: JavaScript、Python、Java、C++ 等
- **标记语言**: Markdown、HTML、XML
- **数据格式**: JSON、YAML、CSV
- **自动检测**: 根据文件扩展名自动识别

#### 🏷️ 多标签页
- **同时编辑**: 打开多个文件，标签页切换
- **标签分组**: 按项目或类型分组管理
- **快速导航**: Ctrl+Tab 快速切换标签

#### 🌙 主题系统
- **浅色主题**: 适合明亮环境
- **深色主题**: 护眼模式，适合夜间使用
- **自定义主题**: 调整颜色、字体大小

#### 💾 自动保存
- **定时保存**: 可配置保存间隔
- **崩溃恢复**: 意外关闭后恢复未保存内容
- **版本历史**: 查看和恢复历史版本

## 基本操作指南

### 启动应用

#### Windows
双击桌面上的「文本编辑器」图标，或在开始菜单中查找。

#### macOS
在应用程序文件夹中找到「文本编辑器」，或从启动台打开。

#### Linux
在应用程序菜单中查找，或使用命令行：
```bash
text-editor
```

### 创建新文件

1. 点击菜单栏 **文件** → **新建**
2. 或使用快捷键 `Ctrl+N` (Windows/Linux) / `Cmd+N` (macOS)
3. 选择文件类型（可选）
4. 开始编辑

### 打开现有文件

1. 点击菜单栏 **文件** → **打开**
2. 或使用快捷键 `Ctrl+O` (Windows/Linux) / `Cmd+O` (macOS)
3. 浏览并选择文件
4. 点击「打开」

### 保存文件

- **保存**: `Ctrl+S` (Windows/Linux) / `Cmd+S` (macOS)
- **另存为**: `Ctrl+Shift+S` (Windows/Linux) / `Cmd+Shift+S` (macOS)

### 编辑文本

#### 基本编辑操作

| 操作 | Windows/Linux | macOS |
|------|---------------|-------|
| 撤销 | Ctrl+Z | Cmd+Z |
| 重做 | Ctrl+Y 或 Ctrl+Shift+Z | Cmd+Shift+Z |
| 剪切 | Ctrl+X | Cmd+X |
| 复制 | Ctrl+C | Cmd+C |
| 粘贴 | Ctrl+V | Cmd+V |
| 全选 | Ctrl+A | Cmd+A |

#### 使用多标签页

- **新建标签**: `Ctrl+T`
- **关闭标签**: `Ctrl+W`
- **切换标签**: `Ctrl+Tab` / `Ctrl+Shift+Tab`
- **跳转到指定标签**: `Alt+1` 到 `Alt+9`

### 搜索与替换

1. 按 `Ctrl+F` 打开搜索框
2. 输入搜索内容
3. 按 `Enter` 查找下一个
4. 按 `Shift+Enter` 查找上一个
5. 点击「替换」标签进行替换操作

### 切换主题

1. 点击菜单栏 **视图** → **主题**
2. 选择「浅色」或「深色」
3. 或使用快捷键 `Ctrl+Shift+L` 快速切换

## 常见问题解答

### Q1: 如何更改默认编码？

**A:** 
1. 点击 **文件** → **设置**
2. 找到「编码」选项
3. 选择默认编码（推荐 UTF-8）
4. 点击「保存」

### Q2: 自动保存如何配置？

**A:**
1. 点击 **文件** → **设置**
2. 找到「自动保存」选项
3. 勾选「启用自动保存」
4. 设置保存间隔（建议 30-60 秒）
5. 点击「保存」

### Q3: 支持哪些编程语言的高亮？

**A:** 支持 50+ 种编程语言，包括：
- JavaScript/TypeScript
- Python
- Java
- C/C++/C#
- Go
- Rust
- PHP
- Ruby
- Swift
- Kotlin
- 以及更多...

### Q4: 如何恢复未保存的文件？

**A:**
1. 重新启动应用
2. 应用会自动检测未保存的文件
3. 在「恢复会话」对话框中选择要恢复的文件
4. 点击「恢复」

### Q5: 可以自定义快捷键吗？

**A:** 
当前版本暂不支持自定义快捷键。该功能计划在 v1.2 版本中添加。

### Q6: 文件最大支持多大？

**A:** 
理论上无限制，但建议单个文件不超过 100MB。过大的文件可能影响性能。

### Q7: 如何在 Dark Mode 下获得更好的体验？

**A:**
1. 使用 **深色主题**（Ctrl+Shift+L）
2. 调整编辑器字体大小（Ctrl+= 放大，Ctrl+- 缩小）
3. 开启「护眼模式」（设置 → 外观 → 护眼模式）

### Q8: 应用崩溃了怎么办？

**A:**
1. 重启应用，会自动恢复未保存的内容
2. 检查日志文件（位置见下方）
3. 如问题持续，请提交 Issue 并附上日志

**日志文件位置:**
- **Windows**: `%APPDATA%\text-editor\logs\`
- **macOS**: `~/Library/Logs/text-editor/`
- **Linux**: `~/.config/text-editor/logs/`

### Q9: 如何反馈问题或建议？

**A:**
- GitHub Issues: [提交问题](https://github.com/your-repo/text-editor/issues)
- 邮件：support@example.com
- 应用内反馈：**帮助** → **反馈问题**

### Q10: 应用是否支持中文？

**A:** 
是的，应用完全支持中文界面和中文内容编辑。默认使用系统语言，可在设置中手动切换。
