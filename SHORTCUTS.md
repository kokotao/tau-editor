# 快捷键速查表

## 文件操作
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+N` | 新建文件 | 创建新的空白标签页 |
| `Ctrl+O` | 打开文件 | 从文件树选择文件打开 |
| `Ctrl+S` | 保存文件 | 保存当前文件到磁盘 |

## 编辑操作
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+Z` | 撤销 | 撤销上一步操作 |
| `Ctrl+Y` | 重做 | 重做已撤销的操作 |
| `Ctrl+A` | 全选 | 选择全部文本内容 |

## 标签页操作
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+Tab` | 下一个标签 | 切换到右侧相邻标签页 |
| `Ctrl+Shift+Tab` | 上一个标签 | 切换到左侧相邻标签页 |
| `Ctrl+W` | 关闭标签 | 关闭当前活动的标签页 |

## 视图与设置
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+,` | 设置面板 | 打开/关闭设置面板 |
| `Ctrl+B` | 切换主题 | 在深色/浅色主题间切换 |

## 字体调整
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl++` | 增大字体 | 字体大小 +1px |
| `Ctrl+-` | 减小字体 | 字体大小 -1px |
| `Ctrl+0` | 重置字体 | 恢复默认 14px |

---

## 提示

- 所有快捷键都支持在编辑区使用
- 在输入框中时，部分快捷键可能会被浏览器拦截
- 字体大小范围：10px - 24px
- 主题切换立即生效，自动保存偏好
