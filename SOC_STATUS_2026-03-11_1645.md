**问题:** Tauri 打包需要 GTK3 开发库，当前环境无 sudo 权限

**解决方案 (需要小纯执行):**
```bash

  libgtk-3-dev libwebkit2gtk-4.1-dev \
  libappindicator3-dev librsvg2-dev patchelf
```

