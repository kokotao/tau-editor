# Markdown 预览阅读主题扩展 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为 Markdown 预览新增 4 套独立阅读主题，并在设置面板与预览区右键菜单中提供切换入口，同时保证设置持久化与现有预览功能不回归。

**Architecture:** 在 `settingsStore` 中增加独立的 `markdownPreviewTheme` 状态，通过设置面板和预览区右键菜单统一更新。`MarkdownPreview.vue` 根据所选主题应用独立 class 和 CSS tokens，避免影响 Monaco 编辑器或全局 `themeSkin`。测试覆盖状态持久化、设置入口、预览渲染与回归行为。

**Tech Stack:** Vue 3、TypeScript、Pinia、Naive UI、Vitest、Vue Test Utils
