## 🔧 测试命令

```bash
# 安装依赖
pnpm install
pnpm exec playwright install

# 运行测试
pnpm test              # 所有测试
pnpm test:unit         # 单元测试
pnpm test:e2e          # E2E 测试
pnpm test:rust         # Rust 测试

# 开发模式
pnpm test:unit:watch   # 单元测试监视
pnpm test:e2e:ui       # E2E UI 模式
pnpm test:e2e:debug    # E2E 调试模式

# 查看报告
pnpm test:unit:coverage  # 覆盖率报告
pnpm test:e2e:report     # E2E 报告
```

## 📊 CI/CD 流程

```yaml
触发条件:
  - push: main, develop
  - pull_request: 所有分支

执行步骤:
  1. Lint (ESLint + Prettier)
  2. TypeCheck (TypeScript)
  3. Unit Tests (Vitest + 覆盖率)
  4. Rust Tests (cargo test + clippy)
  5. E2E Tests (Playwright)
  6. Build (Tauri 构建)
  7. Quality Gate (汇总检查)
```

## 📌 注意事项

1. **测试数据隔离**: 使用临时目录进行测试，避免污染实际文件
2. **Mock 配置**: Tauri API 已全局 Mock，单元测试无需额外配置
3. **浏览器兼容**: E2E 测试覆盖 Chromium、Firefox、WebKit
4. **覆盖率阈值**: 单元测试覆盖率低于 80% 时 CI 失败
5. **并发执行**: 使用 `pool: forks` 加速单元测试执行
