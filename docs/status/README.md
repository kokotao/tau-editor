# Status Docs

历史阶段记录、SOC 调度、构建报告、测试诊断和发布准备文档已统一整理到本目录。

常用文档：

- `PHASE10_BUILD_LOG.md`：Phase 10 构建记录
- `PHASE10_CHECKLIST.md`：阶段验收清单
- `RELEASE_NOTES.md`：发布说明
- `PROJECT_SUMMARY.md`：项目总结
- `E2E_TEST_REPORT.md`：E2E 测试报告
- `TEST_DIAGNOSIS.md`：测试诊断

命名说明：

- `SOC_*`：调度与状态快照
- `PHASE*`：阶段进展与验收
- `PERFORMANCE_*`、`OPTIMIZATION_*`、`UX_*`：专题报告
