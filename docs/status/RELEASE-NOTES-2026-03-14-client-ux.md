# Release Notes - 2026-03-14 (Client UX Upgrade)

## 版本范围

本次发布聚焦客户端体验改造，覆盖关闭稳定性、Markdown 渲染能力、提示降噪、侧栏交互与资源管理器结构优化。

## 主要变更

1. 修复窗口关闭问题
- 修复点击右上角关闭按钮在特定场景无法退出的问题。
- 优化未保存内容关闭确认链路，降低重入阻塞风险。

2. 新增 Markdown 实时预览与 Mermaid
- `.md` 文件支持实时预览。
- 支持 Mermaid 流程图渲染。
- 支持 Markdown 视图模式循环切换：仅编辑 / 分栏 / 仅预览。

3. UI 提示精简
- 去除“编辑器已就绪”启动提示。
- 去除“文件已打开/工作区已打开”等成功提示。
- 保留失败场景错误提示。

4. 左侧面板交互增强
- 支持工具栏按钮折叠/展开侧栏。
- 支持 `Ctrl/Cmd + B` 快捷切换侧栏。
- 支持侧栏宽度拖拽与状态记忆。

5. 资源管理器结构优化
- 目录节点折叠/展开交互优化。
- 文件夹展开/折叠图标状态更明确。
- 支持键盘导航（Enter/Space/Arrow）。
- 工作区刷新后尽量恢复已展开目录状态。
