# ARC-001: 前端组件架构设计

## 1. 组件树结构

```
App.vue
├── Toolbar.vue (工具栏)
│   ├── 文件操作按钮组
│   ├── 编辑操作按钮组
│   └── 视图设置按钮组
├── FileTree.vue (文件树)
│   ├── 文件夹节点 (可折叠)
│   └── 文件节点 (可点击打开)
├── EditorTabs.vue (标签管理)
│   └── EditorTab (单个标签)
│       └── EditorCore.vue (Monaco 编辑器核心)
└── StatusBar.vue (状态栏)
    ├── 光标位置
    ├── 文件编码
    └── 自动保存状态
```

## 2. 组件详细设计

### 2.1 EditorCore.vue (Monaco 封装)

**职责**: 封装 Monaco Editor，提供统一的编辑器接口

**Props**:
```typescript
interface EditorCoreProps {
  modelId: string;           // 编辑器模型 ID
  language?: string;         // 语言模式
  readOnly?: boolean;        // 只读模式
  theme?: string;            // 主题
  options?: EditorOptions;   // Monaco 配置选项
}
```

**Emits**:
```typescript
interface EditorCoreEmits {
  'content-change': (content: string) => void;
  'cursor-change': (position: Position) => void;
  'model-save': () => void;
  'error': (error: Error) => void;
}
```

**暴露的方法**:
```typescript
interface EditorCoreExpose {
  getContent(): string;
  setContent(content: string): void;
  focus(): void;
  layout(): void;
  getSelectedText(): string;
}
```

---

### 2.2 EditorTabs.vue (标签管理)

**职责**: 管理多个编辑器标签页，处理标签切换、关闭等操作

**Props**:
```typescript
interface EditorTabsProps {
  tabs: Tab[];               // 标签列表
  activeTabId: string;       // 当前激活标签 ID
}
```

**Emits**:
```typescript
interface EditorTabsEmits {
  'tab-click': (tabId: string) => void;
  'tab-close': (tabId: string) => void;
  'tab-close-others': (tabId: string) => void;
  'tab-close-all': () => void;
}
```

**数据结构**:
```typescript
interface Tab {
  id: string;
  filePath: string;
  fileName: string;
  isDirty: boolean;          // 是否有未保存的更改
  language: string;
}
```

---

### 2.3 FileTree.vue (文件树)

**职责**: 显示项目文件结构，支持文件夹展开/折叠，文件点击打开

**Props**:
```typescript
interface FileTreeProps {
  rootPath: string;          // 根目录路径
  files: FileEntry[];        // 文件列表
}
```

**Emits**:
```typescript
interface FileTreeEmits {
  'file-open': (filePath: string) => void;
  'folder-toggle': (folderPath: string) => void;
  'context-menu': (entry: FileEntry, event: MouseEvent) => void;
}
```

**数据结构**:
```typescript
interface FileEntry {
  name: string;
  path: string;
  type: 'file' | 'folder';
  children?: FileEntry[];    // 文件夹子项
  isExpanded?: boolean;      // 文件夹是否展开
}
```

---

### 2.4 Toolbar.vue (工具栏)

**职责**: 提供常用操作按钮，如新建、打开、保存、撤销、重做等

**Props**:
```typescript
interface ToolbarProps {
  canUndo: boolean;
  canRedo: boolean;
  isDirty: boolean;
}
```

**Emits**:
```typescript
interface ToolbarEmits {
  'new-file': () => void;
  'open-file': () => void;
  'save': () => void;
  'save-as': () => void;
  'undo': () => void;
  'redo': () => void;
  'toggle-file-tree': () => void;
  'toggle-settings': () => void;
}
```

---

### 2.5 StatusBar.vue (状态栏)

**职责**: 显示编辑器状态信息

**Props**:
```typescript
interface StatusBarProps {
  cursorPosition: { line: number; column: number };
  encoding: string;
  language: string;
  autoSaveEnabled: boolean;
  lastSaveTime?: Date;
}
```

**Emits**:
```typescript
interface StatusBarEmits {
  'encoding-change': (encoding: string) => void;
  'language-change': (language: string) => void;
}
```

## 3. 组件通信流程

```
用户操作
   │
   ▼
Toolbar.vue ─────┬─────> 调用 Store Actions
                 │
FileTree.vue ────┤
                 │
EditorTabs.vue ──┘
                 │
                 ▼
            Pinia Stores
                 │
                 ▼
            Tauri Commands
                 │
                 ▼
            文件系统
```
