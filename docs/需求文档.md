# 文本编辑器需求文档

## 1. 项目概述

构建一个基于 Tauri + React 的跨平台文本编辑器，支持 Markdown 编辑、语法高亮、文件管理等功能。

## 2. 功能需求

### 2.1 核心编辑功能
- [ ] 文本输入与编辑
- [ ] 撤销/重做 (Ctrl+Z / Ctrl+Y)
- [ ] 复制/粘贴/剪切
- [ ] 全选 (Ctrl+A)
- [ ] 查找与替换 (Ctrl+F / Ctrl+H)
- [ ] 行号显示
- [ ] 语法高亮（支持 Markdown、JavaScript、Python 等）

### 2.2 文件管理
- [ ] 新建文件
- [ ] 打开文件（支持多种格式）
- [ ] 保存文件
- [ ] 另存为
- [ ] 最近文件列表
- [ ] 自动保存（可配置间隔）

### 2.3 Markdown 支持
- [ ] Markdown 预览（实时渲染）
- [ ] Markdown 导出（HTML、PDF）
- [ ] 目录导航
- [ ] 表格编辑支持

### 2.4 界面功能
- [ ] 多标签页编辑
- [ ] 分屏编辑
- [ ] 主题切换（亮色/暗色）
- [ ] 字体大小调整
- [ ] 状态栏（显示行列、字数、编码）

### 2.5 设置与偏好
- [ ] 字体设置
- [ ] 主题设置
- [ ] 自动保存间隔
- [ ] 快捷键自定义

## 3. 非功能需求

### 3.1 性能
- 启动时间 < 2 秒
- 大文件（>10MB）打开时间 < 5 秒
- 内存占用 < 200MB（空载）

### 3.2 兼容性
- Windows 10/11
- macOS 11+
- Linux (Ubuntu 20.04+)

### 3.3 安全性
- 文件操作权限控制
- 无远程代码执行风险

## 4. 技术栈

- **前端**: React 18 + TypeScript
- **桌面框架**: Tauri 2.x
- **Rust 后端**: Tauri Commands
- **构建工具**: Vite
- **测试框架**: 
  - 单元测试：Vitest
  - E2E 测试：Playwright
  - Rust 测试：cargo test
