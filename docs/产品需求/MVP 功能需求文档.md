## 1. MVP 概述

### 1.1 产品定位
一款轻量级、跨平台的文本编辑器，填补记事本与 IDE 之间的空白，专注于快速查看和编辑文本/代码文件。

### 1.2 MVP 目标
- **时间目标：** 8 周内完成开发并 release
- **用户目标：** 满足 80% 核心用户的核心需求
- **质量目标：** 崩溃率 < 1%，应用商店评分 4.5+

### 1.3 核心价值主张
| 价值点 | 描述 |
|-------|------|
| 🚀 轻量快速 | 启动<2 秒，内存占用<500MB |
| 📱 跨平台一致 | Windows/macOS/Linux 体验统一 |
| 🎨 语法高亮 | 50+ 语言支持，主题可切换 |
| 📄 多格式支持 | txt/md/json/xml/yaml/csv/html |
| 🔍 高效搜索 | 正则查找替换，多文件搜索 |

- **框架：** Tauri 2.0 (Rust + Web 前端)
- **前端：** React 18 + TypeScript
- **编辑器核心：** CodeMirror 6 / Monaco Editor

## 6. 错误处理

### 6.1 错误类型与处理
| 错误场景 | 错误码 | 用户提示 | 处理方式 |
|---------|-------|---------|---------|
| 文件不存在 | ERR_FILE_NOT_FOUND | "文件不存在，可能已被移动或删除" | 关闭标签 |
| 文件无权限 | ERR_PERMISSION_DENIED | "无法访问文件，请检查权限" | 显示错误 |
| 文件过大 | ERR_FILE_TOO_LARGE | "文件过大（>500MB），可能导致性能问题" | 警告后打开 |
| 保存失败 | ERR_SAVE_FAILED | "保存失败：[具体原因]" | 重试/另存为 |
| 编码错误 | ERR_ENCODING_ERROR | "无法识别文件编码" | 手动选择编码 |
| JSON 格式错误 | ERR_JSON_INVALID | "JSON 格式错误：[位置]" | 高亮错误位置 |

### 6.2 崩溃恢复
- 自动保存内容在崩溃后可恢复
- 崩溃时保存当前打开的文件列表
- 重启时提示恢复未保存的会话
