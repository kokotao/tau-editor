# Tau Editor v0.2.4 发布记录

## 发布概要

- 版本号：`v0.2.4`
- 发布时间：`2026-04-12`
- 发布类型：补丁版本

## 本次变更

- 新增 4 套 Markdown 预览阅读主题：
  - `docs-clean`
  - `paper-soft`
  - `editorial-warm`
  - `graphite-night`
- 新增设置面板中的 Markdown 预览主题切换入口
- 新增 Markdown 预览区右键菜单主题切换入口
- 修复 Markdown 预览右键菜单定位偏移问题，菜单会贴近实际触发位置显示
- 补充主题切换的单元测试与 E2E 回归测试
- 补充右键菜单定位的单元测试与 E2E 回归测试
- 修正与当前设置工作区行为一致的测试断言
